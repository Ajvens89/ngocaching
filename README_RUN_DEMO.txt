MiejskiTrop / NGOcaching — wersja demo bez Supabase

Uruchomienie:
1. npm install
2. npm run build
3. npm run dev

Ta paczka nie wymaga Supabase ani Firebase do pierwszego uruchomienia.
Działa w trybie demo opartym o cookies w przeglądarce.

Konta demo:
- demo@miejski.pl / demo123
- admin@miejski.pl / admin123

Uwagi:
- dane demo zapisują się lokalnie w cookies tej przeglądarki,
- dodawanie punktów z panelu admina działa tylko jako demo w tej przeglądarce,
- to nie jest jeszcze pełna migracja na Firebase.
